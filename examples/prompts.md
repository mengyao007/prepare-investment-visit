# 示例调用

## 只有公司名称

```text
使用 $prepare-investment-visit，为“XX机器人”准备快速版访前材料。
```

## 公司名称与交流对象

```text
调用 $prepare-investment-visit 调研 XX 科技。我们下周与创始人李四交流，
请生成标准版中文 Note，重点关注客户真实性、单位经济性和本轮融资。
```

## 附带 BP

```text
使用 $prepare-investment-visit 阅读附件 BP。
先确认公司主体，再做标准版访前调研；把 BP 中的数字视为公司口径，
并单独列出无法公开核验的关键数据。
```

## 音频简报

```text
使用 $prepare-investment-visit 生成标准版 Note，
再改写成一段约 10 分钟、适合拜访路上收听的中文播客。
```

## 技术型公司深度版

```text
使用 $prepare-investment-visit 调研附件中的气候科技公司，选择 deep 模式。
重点核验技术成熟度、物料与能量平衡、放大风险、CAPEX/OPEX、客户认证和项目融资。
```
