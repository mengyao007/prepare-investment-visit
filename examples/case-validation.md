# 真实案例回归摘要

发布候选版使用两个不同行业、不同研究深度的真实公司案例进行回归。案例材料和生成结果未放入公开发行包；这里只记录不含保密信息的测试方法与结论。

## 原子跳动｜Standard

- 输入：一页公开/可发布 Teaser。
- 输出：6,000 余字访前 Note、材料主张表、证据台账。
- 结果：结构校验 0 错误、0 警告；覆盖产品、行业、竞品、团队、客户、融资、近期进展和 12 个核心问题。
- 关键验证：能区分公司口径、独立来源、推断和待核验事项；不会把规划或宣传口径写成已验证经营事实。

## 碳生万物｜Quick

- 输入：仅公司名称与“拟与创始团队交流”。
- 约束：最多 8 次检索、3,000 字以内。
- 输出：2,995 字访前 Note、8 个直接来源、10 个定制问题。
- 结果：结构校验 0 错误、0 警告；与既有深度版独立得出相同方向判断——团队优势明确，但连续运行、能耗、收率、认证、客户和单位经济性仍待工程证据验证。
- 关键验证：气候科技/工业项目镜头能主动检查物料能量衡算、技术放大、认证、绑定订单和项目融资。

## 边界

案例回归证明工作流和质量门槛可用，不代表相关公司投资价值，也不替代技术、财务或法律尽调。
